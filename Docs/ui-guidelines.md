# UI Guidelines

## Visual Direction

- light background
- crisp separators
- Telegram-like blue tint without modern glass effects
- compact, readable typography
- low-noise surfaces with slightly retro proportions

## Spacing

- dialog rows: fixed height, generous horizontal padding, one-line preview
- chat bubbles: rounded, roomy, but not oversized
- settings: standard grouped table spacing with short labels

## Color Use

- white and soft gray surfaces
- dark text for titles and body copy
- muted gray-blue secondary text
- blue reserved for interaction and unread emphasis
- outgoing message bubble uses a pale blue fill

## Motion

- default UIKit push/pop and table updates only
- no spring-heavy transitions
- no decorative animations on first launch

## Tablet Behavior

- split view for chats when possible
- wider bubble cap on iPad but still constrained for readability
- avoid center-column modern layouts that waste space on older tablets
